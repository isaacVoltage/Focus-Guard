FocusGuard

FocusGuard is a privacy-first, context-aware Manifest V3 web browser extension designed to disrupt impulsive browsing habits by running an ultra-precision local background engine that monitors daily desktop screen time metrics. The application empowers users to regain digital discipline by configuring custom site limits, schedules, and hard time blocks that completely restrict access to specified domains once thresholds are exceeded. To proactively break compulsive behavioral loops, it deploys distinct environment filter profiles like "Study Mode" and "Coding Mode" to instantly isolate problematic networks, alongside a mandatory 15-second "cognitive friction" countdown page when attempting to access distracting channels. All tracked data is compiled into an on-device analytics command center, giving users deep behavioral insights through real-time data visualizations and downloadable daily PDF productivity metrics reports.

Core Features

Web Browser Extension: Built completely on Chrome's modern, high-performance Manifest V3 framework utilizing an isolated local background service worker.
Ultra-Precision Site Tracking: Runs a dedicated local background tracking engine to monitor active browser tab durations down to the exact second.
Daily Limits and Time Blocks: Enforces strict daily time allowances, custom schedules, and absolute lockouts when navigation thresholds are breached.
Environment Context Profiles: Quick-swap specific browser states (such as Study Mode or Coding Mode) to instantly isolate entire categories of distracting domains simultaneously.
Productivity Analysis: Includes a full client-side options dashboard displaying real-time data charts alongside a dedicated on-device PDF generation ledger.


Local Installation Guide

To run this application locally on your machine using Developer Mode:

1. Clone or Download this repository into a dedicated folder on your computer.
2. Open Google Chrome and navigate to your extensions dashboard by entering `chrome://extensions` in the address bar.
3. In the top-right corner of the management window, turn the Developer mode toggle switch ON.
4. Click the Load unpacked button that appears in the top-left section.
5. In the file explorer, choose the main root folder containing your project files (the folder containing `manifest.json`).
6. Click Select Folder. The FocusGuard tile card is now successfully compiled and operational in your browser environment.
